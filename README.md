# Patricio García — CV Repository

This repository contains the bilingual source for my professional CV/resume.

## Files

- `es/cv_es.tex` — Spanish one-page ATS-friendly LaTeX CV.
- `en/cv_en.tex` — English one-page ATS-friendly LaTeX resume.
- `es/README.md` — Spanish Markdown version for recruiters and AI agents.
- `en/README.md` — English Markdown version for recruiters and AI agents.

## Editorial principles

- One-page, reverse-chronological, single-column layout.
- No photo, icons, columns, or visual elements that can hurt ATS parsing.
- Skills are grouped by capability, while experience and projects explain the problems solved with those tools.
- Metrics are included only when they are supported by source material; projected improvements are labeled as projected.

## Build

GitHub Actions compiles both LaTeX files and uploads the generated PDFs as artifacts.
